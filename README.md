# mk-cnwebs-hub

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。项目本身不针对任何特定域名或网站，仅作为静态页面的集中存放与分发平台。所收录的页面均为独立开发或收集整理的内容，不涉及对第三方网站的直接引用或依赖。

## 目录结构

```
mk-cnwebs-hub/
├── pages/          # 存放所有独立 HTML 页面
│   ├── example1/   # 示例页面一（具体页面以实际为准）
│   ├── example2/   # 示例页面二
│   └── ...
├── assets/         # 页面共享资源（CSS、JS、图片等，如有）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- `pages/` 目录下每个子文件夹对应一个独立的 HTML 页面，包含其所需的全部文件。
- `assets/` 目录存放页面间可能共享的静态资源，并非所有页面都依赖此目录。
- 仓库根目录下的其他文件为项目基础设施文件。

## 页面归档说明

本仓库收录的页面遵循以下原则：

- **独立性**：每个页面均为完整可用的 HTML 文档，不依赖外部在线资源（除非特别说明）。
- **无特定指向**：页面内容不针对任何具体域名、网站或商业实体。
- **多样性**：页面主题、用途各异，可能包括工具页面、信息展示、实验性内容等。
- **归档性质**：部分页面可能为历史版本或临时性内容，以当前仓库状态为准。
